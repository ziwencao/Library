本次改动:
实现动态页面
动态页面指的是浏览器在呈现该页面时，数据不是编写该页面时
写死的内容，而是通过服务端处理后的数据。随时间或用户操作后
而产生内容变化的。

实现查看用户列表:访问查看用户列表的Servlet，然后会在浏览器
看到一个表格，列出所有已注册用户信息。
大致操作
Servlet中读取user.dat，将所有用户信息读取出来，并拼接一个
字符串，该字符串内容为HTML代码，并将读取的用户信息也拼接进
去。再将这个字符串转换为一组字节，作为响应正文发送给客户端
